    == Description ==
    POWr Photo Editor is a mobile-responsive, fully-customizable plugin that you can edit **right in the page**.
    Add the widget to your site, or use the shortcode **[powr-photo-editor]** to add Photo Editor to any post.
    <br><br>
    Breathe life into your images without the need for photoshop! Resize, add filters, animations, captions, and watermarks and the subtle borders and shadows that distinguish a professional looking photo from the rest. <br><br>Add CSS3 filters including grayscale, sepia, saturation, hue, invert, opacity, brightness, contrast, and blur. Capture your users attention with an animated entrance. Add custom captions and customize the fonts, colors, and background. Add borders and shadows for a polaroid effect, or reshape the image into a circle or oval. Protect your image by disabling right-click and adding a fully customizable watermark. 
    <br><br>
    The basic version of the POWr Photo Editor is free!

    == Installation ==
    To install POWr Photo Editor
    <br><br>
    1. Upload 'powr-photo-editor.php' to the '/wp-content/plugins/' directory
    <br>
    2. Activate the plugin through the 'Plugins' menu in WordPress
    <br><br>
    To add POWr Photo Editor as a widget, go to **Appearance->Widgets**, and drag POWr Photo Editor to where you'd like it to appear.
    <br><br>
    To add Photo Editor within a post or a page, simply write the shortcode **[powr-photo-editor label='MY LABEL']** (including the brackets) where you'd like your Photo Editor to appear. Then visit your page!
    <br><br>
    Want a more detailed tutorial? Visit [How to add Photo Editor to your Wordpress site](http://www.powr.io/tutorials/how-to-add-photo-editor-plugin-to-your-wordpress-site)


    == Frequently Asked Questions ==
    = How can I add Photo Editor to a blog post or page? = Just add the shortcode **[powr-photo-editor label='MY LABEL']** (including the brackets) in your post where you'd like your Photo Editor to appear. Then visit your page!

    = How can I add  Photo Editor to my site multiple times? = Just be sure to specify different label text, eg **[powr-photo-editor label='MY LABEL 1']**, **[powr-photo-editor label='MY LABEL 2']** for each time you use the plugin

    = Why do you require me to sign up? = Signing up is what allows us to track which plugins are yours, and allows you to edit them right in your page.

    = Will my users see the settings icon in the corner of the Photo Editor? = No! You only see the settings icon because you own the plugins. To temporarily hide it, click on your page and then type 'p + down arrow' (the letter p followed by the down arrow). You can show the icon again by typing 'p + up-arrow'.

    = I can't see the settings icon anymore, what happened? = You are just logged out of POWr.io. You can show the icon again by typing 'p + up-arrow'.

    = Is POWr Photo Editor Free? = Yes, the basic version of Photo Editor is completely free!

    = Why should I upgrade? = Upgrading removes all POWr watermarks, speeds up widget loading speed, gets you premium support, and most importantly supports the creation of more great plugins!
